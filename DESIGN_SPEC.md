# 个人简历网站 - 设计规范文档

## 1. 设计系统概览

**项目定位**：Apple 风格的个人品牌网站 - 极简、高级、内容驱动  
**目标用户**：HR / 招聘经理 / 合作伙伴  
**核心特色**：展示+对话的双重交互体验

---

## 2. 色彩系统

### 主色板
| 名称 | HEX | RGB | 用途 |
|------|-----|-----|------|
| 深黑 | #0A0A0A | 10,10,10 | 正文、主要元素、深色背景 |
| 白色 | #FFFFFF | 255,255,255 | 背景、浅色元素 |
| 浅灰背景 | #FAFAFA | 250,250,250 | Section 背景交替 |
| 纯灰背景 | #F5F5F5 | 245,245,245 | 组件背景 |
| 边框灰 | #EBEBEB | 235,235,235 | 边框、分割线 |
| 文字灰 | #6E6E73 | 110,110,115 | 次级文本 |
| 浅文字灰 | #99989C | 153,152,156 | 弱化文本 |
| 强调蓝 | #2563EB | 37,99,235 | 链接、强调、CTA |
| 深蓝 | #1E40AF | 30,64,175 | 蓝色 Hover 状态 |

### 色彩使用规则
- **深黑 (#0A0A0A)**：Hero 标题、Timeline 节点、按钮背景
- **强调蓝 (#2563EB)**：链接、标签、高亮文字、聊天用户消息
- **灰色系**：背景、边框、次级文本（用 6E6E73 - 99989C 区分层级）

### 阴影预设

| 名称 | Shadow | 用途 |
|------|--------|------|
| 无 | none | 默认 |
| 浅 | 0 1px 3px rgba(0,0,0,0.06), 0 1px 2px rgba(0,0,0,0.04) | 卡片默认 |
| 中 | 0 4px 12px rgba(0,0,0,0.08) | 悬停、占位图 |

## 9. 动效规范

### 9.1 过渡时长
- **快速**：0.15s（微交互）
- **标准**：0.25s（卡片 hover、按钮）
- **缓慢**：0.6s（页面加载入场）

### 9.2 缓动函数
```
cubic-bezier(0.4, 0, 0.2, 1)  /* 标准 */
ease-out                       /* 入场 */
ease-in-out                    /* 循环 */
```

### 9.3 动效清单
- 卡片 Hover：`transform: translateY(-4px)` + 阴影增强
- 按钮 Hover：`transform: translateY(-2px)` + 阴影
- Loading：三个点的上下跳跃动画（dotBounce）
- 入场：淡入 + 微上移（fadeIn）

## 10. 响应式断点

| 名称 | 宽度 | 主要调整 |
|------|------|---------|
| 桌面 | ≥1024px | 完整布局，双列 |
| 平板 | 768-1024px | Hero Gap 减少，Stories Grid 调整 |
| 手机 | <768px | 单列堆叠，Padding 减少，字号缩小 |

### 断点详细规则
- **1024px 以上**：Hero 2列，Section 宽度 980px，Padding 40px
- **768-1024px**：Hero 2列 Gap 40px，About 单列，Stories Grid 1列
- **768px 以下**：Hero 1列堆叠，Section Padding 20px，字号 -4-8px

## 11. 设计思路总结

### 整体审美
✨ **Apple 风格** = 极简 + 高级 + 内容驱动 + 优雅留白

### 核心设计原则
1. **信息优先**：排版清晰，层级明确，不堆砌
2. **颜色克制**：黑白灰 + 蓝色强调，其他色彩最少化
3. **交互反馈**：Hover 状态明确（动画 + 阴影变化）
4. **空间呼吸感**：足够的留白，不紧张

### 品牌色运用
- **深黑 #0A0A0A**：权威、专业、高级感
- **强调蓝 #2563EB**：聚焦、可信、行动号召

### 关键视觉分隔
1. 英雄区：左右信息均衡
2. 聊天区：深黑全宽背景，强调互动价值
3. 故事区：大卡片深黑 + 小卡片白色对比
